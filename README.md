## ‚ú® Project Summary

This project is a fully **functional Digital Event Command Center** designed to centralize and control all school events and announcements in real-time. It features a complete user flow for the entire school community (Parents, Students, and Teachers).

### Core Features

* **Comprehensive Control:** Full control over event creation and announcement posting.
* **Live Interactive Calendar:** A fully working calendar for scheduling and viewing all school activities.
* **Secure Registrations & Ticketing:** Users can sign up for events and receive digital tickets.
* **Persistent Backend (Simple):** Data for events, announcements, and purchased tickets is saved locally in the browser/file system, ensuring persistence between sessions.
* Live Updates (Bonus Feature)
